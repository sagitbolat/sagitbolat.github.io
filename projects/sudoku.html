Sudoku Generator & Solver [C#]

C# · .NET WPF

A fully functioning GUI desktop application for generating, solving, and playing Sudoku puzzles, built with the .NET WPF framework. The algorithm execution and the UI run asynchronously, keeping the application fully responsive while generation or solving is in progress.

Features

  1. Puzzle Generation — A backtracking algorithm generates a complete valid number grid. To improve efficiency, every time a number is placed, it is removed from the candidate lists for all cells in the same row, column, and 3×3 box. Numbers are then removed at random until a puzzle of the specified difficulty is reached.
  2. Guaranteed Single-Solution Puzzles — After each removal, the solver is run on the resulting puzzle. If more than one solution is found, the removal is reverted and a different cell is chosen. After a configurable number of failed attempts, the generator stops to avoid infinite loops.
  3. Puzzle Solver — Uses the same backtracking approach as generation to solve any valid Sudoku puzzle.
  4. Adjustable Calculation Speed — The speed at which the algorithms execute can be slowed down, allowing the user to watch the backtracking process step by step in real time.
  5. Playable UI — The user can attempt to solve the generated puzzle themselves. Navigation uses WASD or arrow keys; number keys input digits; the 0 key erases. Generated cells cannot be modified.
  6. Asynchronous Design — The UI thread and algorithm threads run concurrently, keeping the application responsive at all times. This is what enables live rendering of the generation and solving processes.
